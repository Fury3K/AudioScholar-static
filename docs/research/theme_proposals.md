# Color Theme Proposals: Modern Tech & Nature Calm

## Executive Summary
This document proposes two complete color themes for the AudioScholar application: **"Quantum Indigo"** (Modern/Tech) and **"Zen Sage"** (Nature/Calm). These themes are designed to complement the existing "Classic" theme, offering users distinct aesthetic choices—one vibrant and futuristic, the other grounded and soothing. All colors adhere to Material Design 3 guidelines for accessibility and contrast in both Light and Dark modes.

---

## Theme 1: "Quantum Indigo" (Modern/Tech)
**Vibe:** Futuristic, Deep, Professional, High-Contrast.  
**Base Hue:** Deep Indigo (#4F378B) and Tech Blue.

### Light Mode Palette
| Color Role | Hex Code | Description |
| :--- | :--- | :--- |
| **Primary** | `#4F378B` | Deep indigo, commanding attention. |
| **OnPrimary** | `#FFFFFF` | White text for maximum readability on indigo. |
| **Secondary** | `#3D5C71` | Muted tech-blue, professional and complementary. |
| **OnSecondary** | `#FFFFFF` | White text on secondary elements. |
| **Background** | `#FDFBFF` | Clean, slightly cool white. |
| **OnBackground** | `#1A1C1E` | Almost black, sharp contrast for content. |
| **Surface** | `#FDFBFF` | Matches background for a clean, flat look. |
| **OnSurface** | `#1A1C1E` | Standard dark text. |

### Dark Mode Palette
| Color Role | Hex Code | Description |
| :--- | :--- | :--- |
| **Primary** | `#D0BCFF` | Bright lavender/indigo, glowing against dark bg. |
| **OnPrimary** | `#380D75` | Deep purple text for button labels (high contrast). |
| **Secondary** | `#A5CBE2` | Soft blue-grey, receding but visible. |
| **OnSecondary** | `#0E3147` | Dark blue text on secondary elements. |
| **Background** | `#1A1C1E` | Deep charcoal/black, easy on the eyes. |
| **OnBackground** | `#E3E2E6` | Light grey text, reduced eye strain. |
| **Surface** | `#1A1C1E` | Matches background. |
| **OnSurface** | `#E3E2E6` | Standard light text. |

## Theme 2: "Zen Sage" (Nature/Calm)
**Vibe:** Organic, Restorative, Earthy, Low-Stress.  
**Base Hue:** Moss Green (#4C662B) and Olive.

### Light Mode Palette
| Color Role | Hex Code | Description |
| :--- | :--- | :--- |
| **Primary** | `#4C662B` | Deep moss green, grounding and natural. |
| **OnPrimary** | `#FFFFFF` | White text on green. |
| **Secondary** | `#5D6145` | Olive-grey, subtle earthy tone. |
| **OnSecondary** | `#FFFFFF` | White text. |
| **Background** | `#FDFCF5` | Warm off-white (Cream), mimics paper. |
| **OnBackground** | `#1B1C18` | Dark olive-black, softer than pure black. |
| **Surface** | `#FDFCF5` | Warm surface. |
| **OnSurface** | `#1B1C18` | Dark text. |

### Dark Mode Palette
| Color Role | Hex Code | Description |
| :--- | :--- | :--- |
| **Primary** | `#B1D18A` | Soft sage green, calming in low light. |
| **OnPrimary** | `#1F3701` | Dark green text for buttons. |
| **Secondary** | `#C5CAAC` | Pale olive, harmonic with primary. |
| **OnSecondary** | `#2F331E` | Dark olive text. |
| **Background** | `#1B1C18` | Very dark earthy grey/black. |
| **OnBackground** | `#E4E3DB` | Warm light grey/beige text. |
| **Surface** | `#1B1C18` | Matches background. |
| **OnSurface** | `#E4E3DB` | Standard light text. |
